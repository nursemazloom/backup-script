# h1
# Backup Script with Telegram Notifications

این پروژه شامل یک اسکریپت پایتون برای ایجاد بکاپ از دایرکتوری‌های خاص و ارسال آن به یک چت تلگرام است. همچنین یک اسکریپت برای نصب و پیکربندی خودکار اسکریپت پایتون و تنظیمات کرون جاب برای اجرای خودکار آن هر چند ساعت وجود دارد.

## ویژگی‌ها

+ **ایجاد بکاپ:** اسکریپت به طور خودکار از دایرکتوری‌های مشخص شده بکاپ می‌گیرد و فایل‌های بکاپ را در یک فایل فشرده ذخیره می‌کند.
+ **ارسال بکاپ به تلگرام:** پس از ایجاد بکاپ، فایل بکاپ به یک چت تلگرامی ارسال می‌شود.
+ **پیکربندی خودکار:** با استفاده از اسکریپت خودکار، تمامی پیش‌نیازها نصب شده و اسکریپت پایتون به صورت خودکار پیکربندی می‌شود.
+ **زمان‌بندی خودکار:** امکان تنظیم زمان‌بندی دلخواه برای اجرای اسکریپت پایتون با استفاده از کرون جاب قابل تنظیم.

## پیش‌نیازها

- **Python 3** به همراه `pip`
- کتابخانه‌های `requests` و `jdatetime` (که همگی با استفاده از اسکریپت بصورت اتومات نصب می‌شوند)

## نصب و راه‌اندازی

برای نصب و راه‌اندازی این پروژه مراحل زیر را دنبال کنید:

1. **دانلود اسکریپت نصب و اجرای آن:**
   ابتدا اسکریپت `setup.sh` را دانلود و اجرا کنید:

```bash
   curl -O https://raw.githubusercontent.com/your-username/backup-script/main/setup.sh
   chmod +x setup.sh
   ./setup.sh
```
2. تمام شد حالا از بکاپ خودکار لذت ببرید
