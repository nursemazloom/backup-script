# Backup Script with Telegram Notifications

این پروژه شامل یک اسکریپت پایتون برای ایجاد بکاپ از دایرکتوری‌های خاص و ارسال آن به یک چت تلگرام است. همچنین یک `Bash Script` برای نصب و پیکربندی خودکار اسکریپت پایتون و تنظیمات `cron job` برای اجرای خودکار آن هر چند ساعت وجود دارد.

## ویژگی‌ها

- **ایجاد بکاپ:** اسکریپت به طور خودکار از دایرکتوری‌های مشخص شده بکاپ می‌گیرد و فایل‌های بکاپ را در یک فایل فشرده ذخیره می‌کند.
- **ارسال بکاپ به تلگرام:** پس از ایجاد بکاپ، فایل بکاپ به یک چت تلگرامی ارسال می‌شود.
- **پیکربندی خودکار:** با استفاده از `Bash Script`، تمامی پیش‌نیازها نصب شده و اسکریپت پایتون به صورت خودکار پیکربندی می‌شود.
- **زمان‌بندی خودکار:** امکان تنظیم زمان‌بندی دلخواه برای اجرای اسکریپت پایتون با استفاده از `cron job`.

## پیش‌نیازها

- **Python 3** به همراه `pip`
- کتابخانه‌های `requests` و `jdatetime` (با استفاده از اسکریپت نصب می‌شوند)
